AWS Updates (2023): AWS Lambda Extensions Preview: AWS Lambda now supports a new feature called Extensions, which allows you to augment your Lambda invocation environment. This includes tools for monitoring, security, etc. Amazon EC2 Mac Instances: AWS now offers Mac instances for Amazon EC2, allowing developers to run on-demand macOS workloads in the AWS cloud for the first time. AWS Nitro Enclaves: AWS introduced Nitro Enclaves, a new EC2 capability that makes it easier to securely process highly sensitive data. Amazon Aurora Serverless v2: Amazon Aurora Serverless v2 is now in preview. It scales instantly from hundreds to hundreds-of-thousands of transactions in a fraction of a second.

Azure Updates (2023): Microsoft Fabric event streams: Microsoft Fabric now offers event streams that enable customers to ingest real-time event data from external event sources into the data stores in Fabric. The events can be transformed into various native formats required for the target destination. Azure Event Hubs Updates: Azure Event Hubs launched their Dedicated self-serve clusters capability. Additionally, they announced support for Compaction GA for Kafka and AMQP, Mirror Maker 2 support GA, Kafka Connect support GA, JSON Schema in schema registry for Kafka, Managed Identity for streaming capture, and a Data Generator public preview. Apache Kafka related changes, Managed Identity for streaming capture, JSON Schema in schema registry for Kafka, Data generator public preview. Azure Event Grid: Azure Event Grid now supports bi-directional communication via MQTT version 5. Azure DevOps 2023 Q1: Azure DevOps has delivered multiple features in Q1 2023, including improvements on security and new features based on customer feedback

Advantages of AWS

AWS has a number of advantages which makes it a popular choice among users: User friendly: AWS has a user-friendly platform that makes it easy to use, even for beginners. It comes with very detailed and simple documentation which makes it easier to use the platform. Secure: It is a secure platform that uses IAM so that no unauthorized person can use this platform. AWS has several measures, such as Transport Layer Security certificates and tools with encryption that can combat security threats. Storage: AWS offers high storage that can be used for data backups and archives. It helps in providing storage utilizing a web-based interface as well.
